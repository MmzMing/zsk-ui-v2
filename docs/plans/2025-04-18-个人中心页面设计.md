# 个人中心页面设计方案

## 一、页面整体布局

左右双栏布局：

- **左侧**：用户身份卡片（约280px宽，固定）
- **右侧**：内容区域（flex-1，响应式）

左侧按钮用于**切换右侧内容区**，不是跳转页面。

```
┌─────────────────────────────────────────────────┐
│                   个人中心                       │
├──────────────┬──────────────────────────────────┤
│              │                                  │
│  用户卡片    │         内容区域                 │
│  - 头像      │         - 我的作品               │
│  - 简介      │         - 收藏列表              │
│  - 数据      │         - 关注/粉丝            │
│  - 4个Tab   │         - 设置                 │
│    按钮      │         (Hover Effect)        │
│              │                                  │
└──────────────┴──────────────────────────────────┘
```

## 二、左侧用户卡片（DraggableCard）

### 结构设计

```
┌─────────────────────────┐
│      Avatar 头像         │  ← 大头像（64px）
│         昵称            │  ← 用户名/头衔
│      个人简介           │  ← bio（1-2行）
├─────────────────────────┤
│   关注    粉丝    获赞  │  ← 水平排列
├─────────────────────────┤
│  ✏️  💼  📩  🔒      │  ← 4个Tab按钮
└─────────────────────────┘
```

### 按钮说明（Tab切换）

| 图标       | 功能         | 右侧内容     |
| -------- | ---------- | ---------- |
| ✏️ 编辑信息  | 编辑个人资料   | EditProfile |
| 💼 管理作品  | 作品列表管理   | WorksGrid  |
| 📩 消息管理  | 消息通知管理   | Messages   |
| 🔒 账号安全  | 账号安全设置   | Security   |

### 样式

- 使用 aceternity 的 Single card（DraggableCard）
- 圆角卡片，白色背景
- 统计数据水平排列，黑白配色

## 三、右侧内容区域

### 对应关系

| Tab | 组件 | 说明 |
|-----|------|------|
| 编辑信息 | EditProfile | 个人资料编辑表单 |
| 管理作品 | WorksGrid | Hover Effect 卡片网格 |
| 消息管理 | MessagesPanel | 消息通知列表 |
| 账号安全 | SecurityPanel | 账号安全设置表单 |

### 作品卡片（Hover Effect）

```
┌─────────────────────────┐
│         封面图          │
├─────────────────────────┤
│     作品标题            │
│     简介/描述          │
├─────────────────────────┤
│  👁️  👍  💬        │
└─────────────────────────┘
```

## 四、API 设计

已在 `src/api/profile/index.ts` 创建：

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/profile | 获取用户资料 |
| PUT | /api/profile | 更新用户资料 |
| GET | /api/profile/works | 获取作品列表 |
| DELETE | /api/profile/works/:id | 删除作品 |
| GET | /api/profile/favorites | 收藏列表 |
| DELETE | /api/profile/favorites/:id | 取消收藏 |
| GET | /api/profile/following | 关注列表 |
| GET | /api/profile/followers | 粉丝列表 |
| POST | /api/profile/follow/:userId | 关注用户 |
| DELETE | /api/profile/follow/:userId | 取消关注 |

## 五、组件列表

1. `ProfileCard` - 左侧用户卡片（DraggableCard）
2. `EditProfile` - 个人资料编辑组件
3. `WorksGrid` - 右侧作品网格（Hover Effect）
4. `WorkCard` - 单个作品卡片（Hover Effect）
5. `WorkSkeleton` - 骨架屏
6. `MessagesPanel` - 消息管理面板
7. `SecurityPanel` - 账号安全设置面板

## 六、开发顺序

1. 创建左侧用户卡片组件（带Tab切换）
2. 创建右侧内容区基础布局（4个Tab内容）
3. 创建作品卡片 + 骨架屏
4. 实现各Tab内容组件
5. 添加加载/错误状态
