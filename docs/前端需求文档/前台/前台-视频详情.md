## 数据加载策略（瀑布式）

```
页面挂载
  │
  ├─ 阶段1：GET /docHomeVideo/detail/{id}
  │     → 骨架屏等待 → 渲染视频播放器+视频信息
  │
  ├─ 阶段2（并行）：GET /docHomeVideo/interaction/{id}
  │                 GET /docHomeVideo/collections/{videoId}
  │     → 渲染交互栏+合集侧栏
  │
  └─ 阶段3（懒加载）：GET /docHomeVideo/comments/{videoId}
        → IntersectionObserver 监听，用户滚动到评论区才触发
        → 渲染评论列表
```

## 功能需求

### 1. 面包屑导航
- **功能描述：** 顶部导航栏，支持返回搜索页和面包屑路径展示
- **展示内容：**
  - 返回搜索按钮（← 返回搜索）
  - 面包屑路径：首页 / 搜索 / 视频标题
- **交互行为：** 点击"返回搜索"跳转至 `/search`

### 2. 视频播放
- **功能描述：** 提供视频播放功能，支持播放、暂停、进度控制等
- **播放器功能：**
  - 视频播放/暂停
  - 进度条控制
  - 音量调节
  - 全屏播放（移动端横屏全屏）
  - 播放速度调整
  - 画质选择
  - 播放进度自动保存与恢复（LocalStorage）
  - 音量设置记忆
  - 音频增强
- **技术实现：** 使用 Vidstack 视频播放器（项目封装 `VideoPlayer` 组件）
- **播放器尺寸：** 16:9 比例，左侧全宽展示

### 3. 视频信息
- **功能描述：** 展示视频的详细信息
- **展示内容：**
  - 视频标题（h1，加粗）
  - 视频分类（Chip 标签，primary 色）
  - 视频标签（Chip 标签列表）
  - 观看次数（右侧对齐，带眼睛图标）
  - 视频描述（灰色小字）

### 4. 交互功能
- **功能描述：** 提供视频交互操作和作者信息展示
- **功能包括：**
  - 点赞视频（切换状态，乐观更新，红色心形图标）
  - 收藏视频（切换状态，乐观更新，黄色星形图标）
  - 分享视频（复制链接到剪贴板）
  - 作者头像+昵称+粉丝数
  - 关注作者（切换状态，乐观更新，solid/bordered 按钮切换）
- **未登录处理：** 所有交互操作提示"请先登录"
- **乐观更新策略：** 先更新 UI，请求失败后回滚

### 5. 视频合集
- **功能描述：** 右侧悬浮展示视频所属的公开合集列表
- **展示内容：**
  - 合集封面+名称+视频数量
  - 合集内视频列表（序号+标题+观看数）
  - 当前播放视频高亮（primary 色+播放图标）
- **交互行为：**
  - 点击合集头部展开/收起视频列表
  - 当前视频所在合集默认展开
  - 点击其他视频跳转至对应视频详情页（`/video/:id`）
- **布局差异：**
  - 桌面端：右侧 30% 宽度，`sticky top-20` 悬浮定位
  - 移动端：折叠到评论区上方，全宽展示

### 6. 评论功能
- **功能描述：** B站式二级评论结构，支持发表评论、回复、点赞
- **加载策略：** IntersectionObserver 懒加载，滚动到评论区才触发请求
- **功能包括：**
  - 评论输入框（登录后可用）
  - 发表根评论
  - 回复根评论（显示"回复 @用户名"）
  - 展开/收起回复列表
  - 评论点赞（乐观更新）
  - 热门/最新排序切换
  - 加载更多（分页）
- **评论结构：**
  - 根评论：无 replyTo，replies 为回复列表
  - 回复评论：有 replyTo（被回复用户信息），挂在根评论下
  - 不存在层级嵌套，所有回复统一在根评论下（B站式）

### 7. 回顶滑块
- **功能描述：** 右侧底部回顶拉动条，向下拖拽释放后弹簧回弹并滚动到顶部
- **显示条件：** 页面滚动超过一屏后显示
- **技术实现：** 复用 `ScrollToTopLever` 组件（Framer Motion 弹簧动画）

## 视觉规范

- **无边框无背景框：** 各区域不使用卡片/边框包裹，背景透明
- **区域分割：** 各层之间以 `border-b border-default-200` 细线分割
- **骨架屏：** 加载态使用灰色脉冲动画模拟内容区域
- **主题适配：** 所有颜色使用 HeroUI 语义 token，支持 dark 主题切换

## 对接后端接口

| API 路径 | HTTP 方法 | 功能描述 |
| :--- | :--- | :--- |
| `/api/document/docHomeVideo/detail/{id}` | GET | 获取视频元信息+详情 |
| `/api/document/docHomeVideo/interaction/{id}` | GET | 获取视频交互信息（点赞收藏+作者关注） |
| `/api/document/docHomeVideo/like/{id}` | POST | 切换视频点赞状态 |
| `/api/document/docHomeVideo/favorite/{id}` | POST | 切换视频收藏状态 |
| `/api/document/docHomeVideo/follow/{authorId}` | POST | 切换关注作者状态 |
| `/api/document/docHomeVideo/comments/{videoId}` | GET | 获取视频评论列表（分页，二级结构） |
| `/api/document/docHomeVideo/comment` | POST | 发表/回复视频评论 |
| `/api/document/docHomeVideo/comment/like/{commentId}` | POST | 切换评论点赞状态 |
| `/api/document/docHomeVideo/collections/{videoId}` | GET | 获取视频所属公开合集列表 |
