# 前台-个人中心

## 功能需求

### 1. 编辑信息
- **功能描述：** 编辑用户的个人信息
- **功能包括：**
  - 编辑个人资料（头像、昵称、简介等）
  - 绑定/解绑第三方账号
  - 隐私设置

### 2. 管理作品
- **功能描述：** 管理用户的作品
- **功能包括：**
  - 我的文档列表
  - 我的视频列表
  - 我的工具列表
  - 作品编辑/删除
  - 作品状态管理

### 3. 消息管理
- **功能描述：** 管理用户的消息通知
- **功能包括：**
  - 系统消息
  - 评论通知
  - 点赞通知
  - 关注通知
  - 消息已读/未读

### 4. 账号安全
- **功能描述：** 账号安全设置
- **功能包括：**
  - 修改密码
  - 通知设置
  - 账号安全设置
