# [设计]门户页面

## 需求概述

前台首页 `/` 路由，采用全屏沉浸式 Banner 设计，根据主题模式（深色/浅色）呈现两种截然不同的视觉风格：暗色主题为「狂风暴雨」风格，浅色主题为「风平浪静」风格。整体效果为动态航海场景，包含波浪、小船、闪电、雨滴等动画元素。

## 页面结构

### 第1层：HomeBanner（全屏动态背景）

**布局**：全屏 100vh，绝对定位分层叠加

- **背景层**：
  - 暗色主题：`#0f0f0f`（闪电时闪烁至 `#2a2a2a` 或 `#cfcfcf`）
  - 浅色主题：`bg-gradient-to-b from-white via-gray-100 to-gray-200`

- **SVG 滤镜层**：
  - `paper-texture`：纸张纹理滤镜，模拟粗糙纸张质感
  - `rough-edges`：边缘粗糙滤镜，模拟剪纸边缘不规则效果
  - 暗色主题下叠加纸张纹理（`mix-blend-overlay`）

- **天气效果层（仅暗色主题）**：
  - **闪电效果**：随机生成闪电路径动画，有普通和大闪两种级别
    - 大闪电：`opacity-100`，带 `drop-shadow`，屏幕震动（`animate-shake`）
    - 普通闪电：`opacity-70`，持续 100ms
    - 触发频率：随机 0.5-3.5 秒间隔
  - **雨滴效果**：30 条随机雨滴，倾斜 15 度下落，带透明度渐变
  - **乌云效果**：顶部 SVG 乌云形状，使用 `rough-edges` 滤镜

- **波浪层**：
  - **浅色主题**：4 层涟漪波浪（`ripple` + `normal` 类型）
    - 颜色从 `gray-200` 到 `gray-500` 逐层加深
    - 振幅 5-20px，周期 18-30 秒
    - 方向交替（左/右）
  - **暗色主题**：4 层剧烈波浪（`ripple` + `normal` + `violent` 类型）
    - 颜色从 `#1a1a1a` 到 `#000` 逐层加深
    - 振幅 15-80px，周期 4-20 秒
    - 包含尖刺状巨浪效果（`violent` 类型）

- **小船层**：
  - 海盗船 SVG 图像（`StarDots-2026042403453245646.svg`）
  - 从右向左匀速航行（`110vw` 到 `-120vw`）
  - 暗色主题：40 秒周期，剧烈随机摆动（y: ±40px, rotate: ±20°）
  - 浅色主题：60 秒周期，平缓摆动（y: ±15px, rotate: ±5°）
  - 使用 `useAnimation` 控制不规则摆动

- **内容区域（左上角）**：
  - **标题前缀**：Anchor 图标 + 「航海日志」/「航行记录」标签
  - **主标题**：
    - 暗色主题：「狂风暴雨」（`text-4xl` 到 `text-8xl`，闪烁动画 `opacity: [0.6, 0.8, 0.6]`）
    - 浅色主题：「风平浪静」
  - **副标题**：
    - 暗色主题：「外面风雨交加，不妨静下心来休息，为知识充电」
    - 浅色主题：「在这风平浪静的日子里，正是整理知识、规划学习的好时机」
  - **航海记录统计（VoyageStats）**：
    - 航海时长（网站运行时间，从 API 获取，失败默认 30 天）
    - 船员人数（累计访问人数，从 API 获取，失败默认 12345）
    - 航行状态（固定显示「一帆风顺」）
    - 每个统计项带图标、悬停 tooltip 提示

## 技术方案

### 文件结构

```
src/pages/Home/
├── index.tsx           # 页面主入口，仅渲染 HomeBanner
└── Banner/
    └── index.tsx       # Banner 组件，包含所有动画和视觉效果
```

### 核心组件

| 组件 | 用途 | 技术细节 |
|------|------|----------|
| `TextureFilters` | SVG 滤镜定义 | `feTurbulence` + `feDisplacementMap` 实现纸张纹理和粗糙边缘 |
| `Lightning` | 闪电效果 | `framer-motion` SVG path 动画，随机生成闪电路径 |
| `Rain` | 雨滴效果 | `framer-motion` div 动画，30 条随机雨滴无限下落 |
| `WaveLayer` | 波浪层 | `framer-motion` SVG path 动画，支持 3 种波浪类型（ripple/normal/violent） |
| `Boat` | 小船动画 | `framer-motion` + `useAnimation`，随机不规则摆动 |
| `VoyageStats` | 统计信息 | 从 API 获取数据，带悬停 tooltip |

### 主题适配

- 使用 `useTheme` 获取 `actualTheme`
- 暗色/浅色主题切换时所有动画参数（振幅、周期、颜色）动态变化
- 文字使用 `mix-blend-difference` 确保在暗色主题下可读

### 性能优化

- 雨滴使用 `useMemo` 生成随机数据，避免重复计算
- 波浪路径使用 `useMemo` 预生成
- 闪电使用 `AnimatePresence` 控制挂载/卸载
- 所有动画使用 `framer-motion` 的 GPU 加速

### 交互细节

- 统计项悬停显示 tooltip（标签 + 说明）
- 闪电触发时屏幕震动效果（大闪电）
- 小船持续航行，不受主题切换影响
