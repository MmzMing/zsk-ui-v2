## 0. 页面布局

### 0.1 整体布局
- **布局类型：** 工具栏 + 表格布局
- **顶部工具栏区域：**

  - 支持多选批量操作（批量删除、批量上架/下架、批量迁移分类）

- **弹窗组件：**
  - 新增/编辑文档弹窗
    - 上方：文章图片区域（悬停显示"点击修改"或"上传"）
    - 下方：文档信息表单
  - 图片裁剪弹窗（使用 `@/components/ui/image-crop`）

- 移动端：隐藏部分非关键列，操作列自适应

## 4. 图片上传流程

### 4.1 图片上传/修改流程
1. 用户点击编辑按钮，弹出编辑弹窗
2. 鼠标悬停在图片区域，显示"点击修改"或"上传"提示
3. 用户点击后：
   - 若选择上传：先弹出上传弹窗获取后图片再弹出裁剪弹窗，需要规范图片大小3M以下
   - 若选择修改：直接弹出图片裁剪弹窗
4. 用户在裁剪弹窗中完成操作并确认，然后关闭弹窗，清空 `coverFileId` 参数，开始调用文件上传接口 `/api/document/docFiles/upload`，图片下方显示上传进度
5. 图片上传完成后，将文件ID放入 `coverFileId` 参数，后端返回文件ID，前端保存该ID
6. 下方文档信息提交时，如果没有这个ID则不能点击确认提交

### 4.2 图片状态管理
- 封面图片字段：`coverFileId`（文件ID）、`cover`（文件URL）
- 仅当用户确认修改或上传图片时，清空原有ID并更新为新ID
- 提交按钮需校验 `coverFileId` 是否存在（必填项）

## 5. 接口依赖分析

| 功能 | 后端API | 状态 | 说明 |
|------|---------|------|------|
| 查询文档列表 | GET /api/document/docNote/list | 已支持 | 支持noteName、status、auditStatus、broadCode筛选 |
| 分页查询文档列表 | GET /api/document/docNote/page | 已支持 | 支持分页参数 |
| 获取文档详情 | GET /api/document/docNote/{id} | 已支持 | 根据ID获取文档详情 |
| 新增文档 | POST /api/document/docNote | 已支持 | 完整支持，需传入coverFileId |
| 修改文档 | PUT /api/document/docNote | 已支持 | 完整支持 |
| 删除文档 | DELETE /api/document/docNote/{ids} | 已支持 | 支持批量删除 |
| 批量更新状态 | PUT /api/document/docNote/status/batch | 已支持 | 支持批量上架/下架 |
| 批量迁移分类 | PUT /api/document/docNote/category/batch | 已支持 | 支持批量修改分类 |
| 切换置顶状态 | PUT /api/document/docNote/{id}/pinned | 已支持 | 切换置顶 |
| 切换推荐状态 | PUT /api/document/docNote/{id}/recommended | 已支持 | 切换推荐 |
| 上传文件 | POST /api/document/docFiles/upload | 已支持 | 上传封面图片，返回文件信息 |
