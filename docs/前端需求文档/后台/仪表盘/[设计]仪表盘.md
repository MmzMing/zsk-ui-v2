## 第一层：概览区

### 1.1 左侧区域 (宽度 70%)

#### 1.1.1 欢迎语卡片

- **功能描述：** 根据当前时间段动态显示欢迎语和对应图标
- **时段规则：**

| 时段 | 时间范围 | 欢迎语 | 图标 (lucide-react) |
|:---|:---|:---|:---|
| 早晨 | 06:00 - 08:59 | 早上好，{userName} | `Sunrise` |
| 上午 | 09:00 - 11:59 | 上午好，{userName} | `Sun` |
| 中午 | 12:00 - 13:59 | 中午好，{userName} | `CloudSun` |
| 下午 | 14:00 - 17:59 | 下午好，{userName} | `Sunset` |
| 晚上 | 18:00 - 22:59 | 晚上好，{userName} | `Moon` |
| 深夜 | 23:00 - 05:59 | 夜深了，{userName}，注意休息 | `MoonStar` |

- **UI 细节：**
  - `userName` 从 `useUserStore` 获取当前登录用户昵称
  - 欢迎语使用 `text-2xl font-bold`，图标放在文字左侧，大小 32px
  - 卡片使用 HeroUI `Card`，`bg-content1`
  - 可附带一行副标题，如 "今天是 2026年4月23日，星期三"

#### 1.1.2 数据统计卡片 (4 个)

- **数据来源：** `GET /dashboard/overview`
- **接口响应结构：**

```typescript
interface DashboardItem {
  key: 'users' | 'docs' | 'videos' | 'comments';
  label: string;   // 显示标签
  value: string;   // 当前数值
  delta: string;   // 变化量，如 "+12.5%"
  description: string; // 描述说明
}
```

- **卡片布局：** 4 列等分网格 (`grid grid-cols-4 gap-4`)，移动端自适应 `grid-cols-2`
- **每张卡片展示：**
  - 图标（根据 `key` 映射）：
    - `users` → `Users` (lucide-react)
    - `docs` → `FileText` (lucide-react)
    - `videos` → `Video` (lucide-react)
    - `comments` → `MessageSquare` (lucide-react)
  - 标签 (`label`)
  - 数值 (`value`)，使用 `text-3xl font-bold` 突出
  - 变化量 (`delta`)：正数绿色 `text-success`，负数红色 `text-danger`，带向上/向下箭头图标
  - 描述 (`description`) 用 `text-small text-default-500` 展示
- **交互：** 鼠标悬浮卡片有轻微放大效果 (`hover:scale-[1.02] transition-transform`)
- **加载态：** 接口请求中使用 HeroUI `Skeleton` 占位

## 第二层：任务管理区 (Gantt)

### 2.1 技术选型

- **组件库：** `@svar-ui/react-gantt`（包名 `wx-react-gantt`）
- **安装：**

```bash
npm install wx-react-gantt
```

- **样式引入：**

```typescript
import "wx-react-gantt/dist/gantt.css";
```

### 2.2 功能需求

#### 2.2.1 任务数据 (Mock)

后端接口未开发，暂用前端 Mock 数据，数据结构遵循 `wx-react-gantt` 要求：

```typescript
interface GanttTask {
  id: number;
  text: string;
  start: Date;
  end: Date;
  duration: number;
  progress: number;
  type: 'task' | 'milestone' | 'summary';
  parent?: number;
}

interface GanttLink {
  id: number;
  source: number;
  target: number;
  type: 's2s' | 's2e' | 'e2s' | 'e2e';
}

interface GanttScale {
  unit: 'year' | 'quarter' | 'month' | 'week' | 'day' | 'hour';
  step: number;
  format: string;
}
```

#### 2.2.2 核心交互

| 交互 | 说明 |
|:---|:---|
| **拖拽移动** | 在时间线上拖拽任务条调整起止时间 |
| **双击编辑** | 双击任务打开编辑弹窗，修改名称、时间、进度等 |
| **工具栏** | 顶部 Toolbar，包含：新增任务、删除任务、上移、下移 |
| **右键菜单** | ContextMenu，包含：新增子任务/上方/下方任务、编辑、删除、移动、缩进/取消缩进 |

#### 2.2.3 工具栏 (Toolbar)

使用 `wx-react-gantt` 内置 `Toolbar` 组件：

```tsx
import { Gantt, Toolbar } from "wx-react-gantt";
import "wx-react-gantt/dist/gantt.css";

<Toolbar api={apiRef.current} />
<Gantt apiRef={apiRef} tasks={tasks} links={links} scales={scales} />
```

#### 2.2.4 右键菜单 (ContextMenu)

使用 `wx-react-gantt` 内置 `ContextMenu` 组件：

```tsx
import { Gantt, ContextMenu } from "wx-react-gantt";

<ContextMenu api={apiRef.current}>
  <Gantt apiRef={apiRef} tasks={tasks} links={links} scales={scales} />
</ContextMenu>
```

默认菜单项包括：Add (Child/Above/Below)、Convert to、Edit、Cut、Copy、Paste、Move (Up/Down)、Indent/Outdent、Delete。

#### 2.2.5 任务编辑器

双击任务默认弹出内置编辑器，如需自定义可通过 `api.intercept("show-editor", ...)` 拦截并替换。

### 2.3 UI 细节

- Gantt 区域使用 HeroUI `Card` 包裹，卡片标题 "任务管理"
- Gantt 组件高度建议 `500px`，可根据实际内容调整
- Gantt 样式需覆盖适配暗色主题（在 `global.css` 中通过 `.dark` 选择器覆盖 `wx-react-gantt` 的 CSS 变量）
