# 后台-系统日志

**文件位置：** `src/pages/Admin/Monitor/SysLog.tsx`
**最后更新日期：** 2026-04-22

---

## 0. 页面布局

### 0.1 整体布局
- **布局类型：** 工具栏 + 表格布局
- **顶部工具栏区域：**
  - 左侧：搜索框（支持操作人模糊搜索）
  - 中间：筛选条件（分类、请求方式、操作状态、业务类型、时间范围）
  - 右侧：操作按钮（批量删除）
- **中部表格区域：** 系统日志列表表格
  - 表格列：分类（category）、操作人（operator）、动作名称（action）、详细描述（detail）、请求方式（requestMethod）、请求URL（requestUrl）、操作状态（status）、消耗时间（costTime）、操作IP（operIp）、创建时间（createdAt）、操作
  - 支持多选批量操作（批量删除）
  - 底部：分页控件（页码、每页大小、总条数）
- **弹窗组件：**
  - 日志详情弹窗（查看请求参数、响应结果等完整信息）
  - 批量删除确认弹窗

### 0.2 响应式适配
- 桌面端：完整展示所有表格列
- 移动端：隐藏请求URL、消耗时间列，操作列自适应

---

## 1. 日志列表

- **功能描述：** 展示系统管理日志列表，支持分页、搜索、筛选
- **展示内容：**
  - 分类（content/user/system）
  - 操作人
  - 动作名称
  - 详细描述
  - 请求方式
  - 请求URL
  - 操作状态（0正常 1异常）
  - 消耗时间(ms)
  - 操作IP
  - 创建时间

---

## 2. 日志查询

- **功能描述：** 支持多条件查询管理日志
- **查询条件：**
  - 操作人（模糊匹配）
  - 分类（content/user/system）
  - 请求方式（GET/POST/PUT/DELETE）
  - 操作状态（0正常 1异常）
  - 模块标题（模糊匹配）
  - 业务类型（0其它 1新增 2修改 3删除 4授权 5导出 6导入）
  - 请求URL（模糊匹配）
  - 操作时间范围

---

## 3. 日志详情

- **功能描述：** 查看日志详细信息
- **展示内容：**
  - 日志ID
  - 分类
  - 操作人
  - 动作名称
  - 详细描述
  - 请求方式
  - 请求URL
  - 请求参数（完整JSON）
  - 响应结果（完整JSON）
  - 操作状态
  - 消耗时间(ms)
  - 操作IP
  - 创建时间

---

## 4. 日志管理

- **功能描述：** 管理系统日志
- **功能包括：**
  - **查看日志详情**：查看日志完整信息
  - **删除日志**：支持单个删除和批量删除
