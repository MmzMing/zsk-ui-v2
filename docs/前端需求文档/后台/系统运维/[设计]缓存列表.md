# [设计]缓存列表

**文件位置：** `src/pages/Admin/Monitor/Cache.tsx`  
**最后更新日期：** 2026-04-22

---

## 0. 页面布局

### 0.1 整体布局
- **布局类型：** 标签页 + 工具栏 + 表格布局
- **顶部标签页区域：**
  - 缓存概览：展示缓存实例和统计信息
  - 缓存键列表：展示缓存键名列表
- **顶部工具栏区域：**
  - 左侧：搜索框（支持缓存键名模糊搜索）
  - 中间：缓存名称筛选下拉框
  - 右侧：操作按钮（刷新缓存、缓存预热、清空全部）
- **中部表格区域：** 缓存键列表表格
  - 表格列：缓存键名（cacheKey）、缓存值摘要（valueSummary）、缓存类型（type）、过期时间（expireTime）、剩余时间（remainingTime）、操作
  - 支持多选批量操作（批量删除、批量刷新）
  - 底部：分页控件（页码、每页大小、总条数）
- **弹窗组件：**
  - 缓存详情弹窗（查看完整缓存值）
  - 批量操作确认弹窗（批量删除、批量刷新）

### 0.2 响应式适配
- 桌面端：完整展示所有表格列
- 移动端：隐藏缓存类型列，操作列自适应

---

## 1. 缓存概览

- **功能描述：** 展示系统缓存整体信息
- **展示内容：**
  - 缓存键总数
  - 缓存命中率
  - QPS
  - Redis服务器信息（版本、运行模式、运行天数、内存使用量）
  - 缓存分布饼图
  - 内存使用仪表盘

---

## 2. 缓存键列表

- **功能描述：** 展示缓存键列表，支持搜索和筛选
- **功能包括：**
  - 分页展示缓存键列表
  - 按缓存键名模糊搜索
  - 按缓存名称筛选
  - 列表字段：缓存键名、缓存值摘要、缓存类型、过期时间、剩余时间
  - 支持多选批量操作

---

## 3. 缓存管理

- **功能描述：** 管理系统缓存
- **功能包括：**
  - **查看缓存详情**：查看缓存键完整信息
  - **删除缓存键**：支持单个删除和批量删除
  - **刷新缓存键**：支持单个刷新和批量刷新
  - **刷新缓存过期时间**：更新缓存TTL
  - **清空指定缓存**：按缓存名称清空
  - **清空所有缓存**：清空全部缓存
  - **缓存预热**：执行缓存预热

---

## 4. 缓存详情

- **功能描述：** 查看缓存详细信息
- **展示内容：**
  - 缓存键名
  - 缓存值（完整内容）
  - 缓存类型
  - 过期时间
  - 创建时间

---

## 5. 接口依赖分析

| 功能 | 后端API | 状态 | 说明 |
|------|---------|------|------|
| 获取缓存日志 | GET /api/system/cache/logs | 已支持 | 获取缓存操作日志 |
| 获取缓存分布 | GET /api/system/cache/distribution | 已支持 | 获取缓存分布饼图数据 |
| 获取内存使用 | GET /api/system/memory/usage | 已支持 | 获取内存使用仪表盘数据 |
| 获取Redis信息 | GET /api/system/cache/info | 已支持 | 获取Redis服务器信息 |
| 获取缓存键名列表 | GET /api/system/cache/keys | 已支持 | 支持cacheName、keyword筛选和分页 |
| 获取缓存信息列表 | GET /api/system/cache/list | 已支持 | 获取缓存信息列表 |
| 获取缓存详细信息 | GET /api/system/cache/info/{cacheKey} | 已支持 | 根据缓存键获取详细信息 |
| 获取缓存值 | GET /api/system/cache/value/{cacheKey} | 已支持 | 获取缓存键对应的值 |
| 判断缓存是否存在 | GET /api/system/cache/exists/{cacheKey} | 已支持 | 判断缓存键是否存在 |
| 获取缓存统计信息 | GET /api/system/cache/statistics | 已支持 | 获取缓存统计数据 |
| 刷新缓存键 | POST /api/system/cache/keys/refresh | 已支持 | 刷新单个缓存键 |
| 批量刷新缓存键 | POST /api/system/cache/keys/batchRefresh | 已支持 | 批量刷新缓存键 |
| 批量删除缓存键 | POST /api/system/cache/keys/batchDelete | 已支持 | 支持批量删除 |
| 缓存预热 | POST /api/system/cache/warmup | 已支持 | 执行缓存预热 |
| 刷新缓存过期时间 | PUT /api/system/cache/refreshTtl | 已支持 | 刷新单个缓存TTL |
| 批量刷新缓存过期时间 | PUT /api/system/cache/refreshTtlBatch | 已支持 | 批量刷新缓存TTL |
| 删除缓存键 | DELETE /api/system/cache/keys/{key} | 已支持 | 删除单个缓存键 |
| 删除缓存 | DELETE /api/system/cache/delete | 已支持 | 删除缓存 |
| 清空指定名称缓存 | DELETE /api/system/cache/clear/{cacheName} | 已支持 | 清空指定名称的缓存 |
| 清空所有缓存 | DELETE /api/system/cache/clearAll | 已支持 | 清空所有缓存 |
