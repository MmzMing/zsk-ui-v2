# 后台-用户行为

**文件位置：** `src/pages/Admin/Ops/UserBehavior.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 在线用户列表

- **功能描述：** 展示当前在线用户列表
- **展示内容：**
  - 用户信息
  - 登录时间
  - 登录IP
  - 登录地点
  - 浏览器
  - 操作系统
  - 在线时长

### 2. 在线用户管理

- **功能描述：** 管理在线用户
- **功能包括：**
  - 强制下线用户
  - 批量强制下线
  - 刷新在线用户列表
  - 搜索在线用户



### 1. 用户行为列表

- **功能描述：** 展示用户行为记录列表，支持分页、搜索、筛选
- **展示内容：**
  - 用户信息
  - 行为类型
  - 行为内容
  - 行为时间
  - 行为 IP
  - 行为地点

### 2. 行为查询
- **功能描述：** 支持多条件查询用户行为
- **查询条件：**
  - 用户名称
  - 行为类型
  - 行为时间范围
  - 行为 IP

### 3. 行为详情
- **功能描述：** 查看用户行为详细信息
- **详情内容：**
  - 行为基本信息
  - 用户信息
  - 行为内容
  - 请求参数
  - 响应结果
  - 设备信息

### 4. 行为统计
- **功能描述：** 用户行为统计分析
- **统计内容：**
  - 用户行为趋势
  - 热门行为类型
  - 活跃用户排行
  - 行为时段分布
  - 数据导出
