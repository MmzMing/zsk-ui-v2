## 0. 页面布局

### 0.1 整体布局
- **布局类型：** 工具栏 + 表格布局
- **顶部工具栏区域：**
  - 左侧：搜索框（支持合集名称模糊搜索）
  - 中间：状态下拉筛选框（正常/下架）、删除状态下拉筛选框（未删除/已删除）

  - 支持多选批量操作（批量删除）
  - 底部：分页控件（页码、每页大小、总条数）
- **弹窗组件：**
  - 合集新增/编辑弹窗（表单编辑合集信息）
  - 合集详情弹窗（展示合集内视频列表，支持排序、移除视频）
  - 添加视频到合集弹窗（从视频列表中选择视频加入合集）
  - 批量删除确认弹窗

### 0.2 响应式适配
- 桌面端：完整展示所有表格列
- 移动端：隐藏部分非关键列（描述、更新时间），操作列自适应

## 3. 合集详情与视频管理

### 3.1 合集新增/编辑弹窗

- **功能描述：** 点击新增或编辑按钮，弹窗展示并编辑合集信息
- **弹窗布局：** 单列表单布局

#### 3.1.1 表单区域

**基本信息表单：**
- **合集名称** (`collectionName`)：必填，文本输入框，最大长度 100 字符
- **合集描述** (`description`)：选填，多行文本域，最大长度 500 字符
- **合集封面** (`coverImage`)：选填，图片上传组件
  - **展示方式：** 图片预览卡片，默认展示当前封面 `coverImage`
  - **空状态：** 显示"点击上传封面"占位区域，带上传图标
  - **悬停交互：** 鼠标悬停显示"点击修改"或"重新上传"遮罩层
  - **上传流程：**
    1. 点击封面区域，弹出文件选择器（限制图片格式：jpg、png、jpeg，大小 3MB 以下）
    2. 选择图片后，先弹出图片裁剪弹窗（使用 `@/components/ui/image-crop`）
    3. 裁剪确认后，调用文件上传接口 `/api/document/docFiles/upload`
    4. 上传过程中显示进度条
    5. 上传完成后，将返回的 `fileUrl` 赋值给 `coverImage`，并更新预览图
  - **删除功能：** 支持删除已上传封面，删除后清空 `coverImage`，恢复空状态

**状态控制：**
- **状态** (`status`)：开关切换，正常/下架（默认正常）

**底部操作按钮：**
- **确认保存** (`POST /api/document/docVideo/collection` 或 `PUT /api/document/docVideo/collection`)：保存合集信息
- **取消：** 关闭弹窗，不保存修改

#### 3.1.2 表单校验规则

| 字段 | 校验规则 |
|------|----------|
| collectionName | 必填，长度 1-100 字符 |
| description | 选填，长度不超过 500 字符 |
| coverImage | 选填，必须为合法图片 URL |
| status | 选填，默认 1（正常） |

#### 3.1.3 数据回显与初始化

- 编辑模式：根据合集 ID 调用 `GET /api/document/docVideo/collection/{id}` 获取详情，回显所有字段
- 新增模式：所有字段为空，状态默认值为 `status=1`（正常）

### 3.2 合集详情弹窗

- **功能描述：** 点击详情按钮，弹窗展示合集基本信息及合集内视频列表
- **弹窗布局：** 上下分栏布局，上方为合集信息展示区，下方为视频列表表格区

#### 3.2.1 合集信息展示区

- **合集封面：** 左侧展示封面图片，点击可预览大图
- **合集信息：** 右侧展示合集名称、合集描述、状态、创建时间、更新时间
- **操作按钮：**
  - **编辑合集：** 打开合集编辑弹窗
  - **添加视频：** 打开添加视频到合集弹窗

#### 3.2.2 视频列表表格区

- **表格列：** 视频封面、视频标题、作者、分类、排序顺序、添加时间、操作
- **排序功能：** 支持拖拽排序或上下箭头调整顺序
  - 拖拽排序：长按视频行拖拽调整顺序，拖拽完成后调用 `PUT /api/document/docVideo/collection/{collectionId}/videos/order` 更新排序
  - 箭头排序：每行提供"上移"/"下移"按钮，点击后调整顺序并保存
- **移除视频：** 点击移除按钮，确认后调用 `DELETE /api/document/docVideo/collection/{collectionId}/videos/{videoId}` 从合集中移除视频
- **视频详情：** 点击视频标题或封面，可跳转至视频详情页或打开视频预览弹窗

### 3.3 添加视频到合集弹窗

- **功能描述：** 点击添加视频按钮，弹窗展示可选视频列表，支持搜索和选择
- **弹窗布局：** 顶部搜索区 + 中部视频列表区 + 底部操作区

#### 3.3.1 搜索区

- **视频标题搜索：** 支持模糊搜索视频标题
- **分类筛选：** 按大类/小类筛选视频
- **搜索按钮：** 点击后刷新视频列表

#### 3.3.2 视频列表区

- **展示方式：** 卡片网格或表格列表，展示视频封面、视频标题、作者、分类
- **选择方式：** 支持单选或多选（复选框）
- **已加入提示：** 已在当前合集中的视频显示"已添加"标识，不可重复选择
- **分页：** 支持分页加载，避免一次性加载过多视频

#### 3.3.3 底部操作区

- **确认添加：** 调用 `POST /api/document/docVideo/collection/{collectionId}/videos` 将选中的视频添加到合集
  - 请求体：`{ videoId: "1", sortOrder: 1 }`，`sortOrder` 默认为当前合集视频数量 + 1
- **取消：** 关闭弹窗，不添加视频

### 4.1 合集状态
- **功能描述：** 在列表中直接点击上架/下架按钮切换合集状态

## 5. 接口依赖分析

| 功能 | 后端API | 状态 | 说明 |
|------|---------|------|------|
| 查询合集列表 | GET /api/document/docVideo/collection/list | 已支持 | 支持collectionName、status、deleted筛选 |
| 分页查询合集列表 | GET /api/document/docVideo/collection/page | 已支持 | 支持分页参数pageNum、pageSize |
| 获取合集详情 | GET /api/document/docVideo/collection/{id} | 已支持 | 根据ID获取合集详情（含视频列表） |
| 新增合集 | POST /api/document/docVideo/collection | 已支持 | 完整支持 |
| 修改合集 | PUT /api/document/docVideo/collection | 已支持 | 完整支持 |
| 删除合集 | DELETE /api/document/docVideo/collection/{ids} | 已支持 | 支持批量删除 |
| 获取合集内视频列表 | GET /api/document/docVideo/collection/{collectionId}/videos | 已支持 | 获取指定合集内的视频列表 |
| 添加视频到合集 | POST /api/document/docVideo/collection/{collectionId}/videos | 已支持 | 将视频添加到指定合集 |
| 从合集中移除视频 | DELETE /api/document/docVideo/collection/{collectionId}/videos/{videoId} | 已支持 | 从指定合集中移除视频 |
| 更新合集内视频排序 | PUT /api/document/docVideo/collection/{collectionId}/videos/order | 已支持 | 更新合集内视频的排序顺序 |

## 6. 数据结构说明

### 6.1 合集列表项

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 合集ID |
| userId | string | 用户ID |
| collectionName | string | 合集名称 |
| description | string | 合集描述 |
| coverImage | string | 封面图片URL |
| status | number | 状态（1-正常，2-下架） |
| deleted | number | 删除标记（0-未删除，1-已删除） |
| createTime | string | 创建时间 |
| updateTime | string | 更新时间 |

### 6.2 合集详情项（含视频列表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 合集ID |
| userId | string | 用户ID |
| collectionName | string | 合集名称 |
| description | string | 合集描述 |
| coverImage | string | 封面图片URL |
| status | number | 状态 |
| deleted | number | 删除标记 |
| createTime | string | 创建时间 |
| updateTime | string | 更新时间 |
| videos | array | 合集内视频列表（结构同视频列表响应） |

### 6.3 合集表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 编辑时必填 | 合集ID |
| userId | string | 是 | 用户ID |
| collectionName | string | 是 | 合集名称 |
| description | string | 否 | 合集描述 |
| coverImage | string | 否 | 封面图片URL |
| status | number | 否 | 状态（默认1） |

### 6.4 合集视频关联字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | string | 是 | 视频ID |
| sortOrder | number | 否 | 排序顺序（默认0） |

### 6.5 视频排序更新字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoOrders | array | 是 | 视频排序列表 |
| videoOrders[].videoId | string | 是 | 视频ID |
| videoOrders[].sortOrder | number | 是 | 排序顺序 |
