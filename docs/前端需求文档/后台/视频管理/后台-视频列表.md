## 3. 视频详情与操作

### 3.1 视频元信息编辑弹窗

- **功能描述：** 点击元信息按钮，弹窗展示并编辑视频详细信息，支持上传封面图片、上传/预览视频
- **弹窗布局：** 左右分栏布局，左侧为媒体上传区域，右侧为元信息表单区域

#### 3.1.1 左侧媒体区域（占比约 40%）

**上方：封面图片上传区**
- **展示方式：** 图片预览卡片，默认展示当前视频封面 `videoFile.thumbnail.fileUrl`
- **空状态：** 显示"点击上传封面"占位区域，带上传图标
- **悬停交互：** 鼠标悬停显示"点击修改"或"重新上传"遮罩层
- **上传流程：**
  1. 点击封面区域，弹出文件选择器（限制图片格式：jpg、png、jpeg，大小 3MB 以下）
  2. 选择图片后，先弹出图片裁剪弹窗（使用 `@/components/ui/image-crop`）
  3. 裁剪确认后，调用文件上传接口 `/api/document/docFiles/upload`
  4. 上传过程中显示进度条
  5. 上传完成后，将返回的 `fileId` 赋值给 `coverFileId`，并更新预览图
  6. 提交表单时必须校验 `coverFileId` 是否存在
- **删除功能：** 支持删除已上传封面，删除后清空 `coverFileId`，恢复空状态

**下方：视频上传与预览区**
- **视频预览：** 使用项目封装的 `VideoPlayer` 组件（`@vidstack/react`）播放当前视频 `videoFile.video.fileUrl`
- **空状态：** 显示"点击上传视频"占位区域，带上传图标
- **上传流程：**
  1. 点击上传区域，弹出文件选择器（限制视频格式：mp4、mov、avi，大小 500MB 以下）
  2. 选择视频后，调用视频上传接口 `POST /api/document/docVideo/upload` 或文件上传接口
  3. 上传过程中显示进度条和上传百分比
  4. 上传完成后，将返回的 `fileId` 赋值给 `fileId`，并更新视频预览
  5. 提交表单时必须校验 `fileId` 是否存在
- **重新上传：** 支持删除已有视频后重新上传，删除后清空 `fileId`
- **视频信息展示：** 上传成功后，自动解析并展示视频元数据（如分辨率、时长等，若后端返回则展示）

#### 3.1.2 右侧元信息表单区域（占比约 60%）

**基本信息表单：**
- **视频标题** (`videoTitle`)：必填，文本输入框，最大长度 100 字符
- **视频描述** (`fileContent`)：选填，多行文本域，最大长度 2000 字符
- **标签** (`tags`)：选填，标签输入组件，支持多标签，逗号分隔，最多 10 个标签
- **分类选择：**
  - **大类** (`broadCode`)：必填，下拉选择框，从字典接口获取大类列表
  - **小类** (`narrowCode`)：选填，下拉选择框，根据大类联动加载小类列表

**元数据表单：**
- **元数据** (`metaData`)：选填，JSON 格式文本域，用于存储视频技术参数（如分辨率、码率、时长等）
- **分集信息** (`videoDtl`)：选填，动态表单数组，支持添加/删除分集
  - 每行分集包含：分集标题输入框、分集视频地址输入框、分集时长输入框
  - 支持拖拽排序（可选）

**状态控制：**
- **审核状态** (`auditStatus`)：只读展示，不可编辑（由审核流程控制）
- **发布状态** (`status`)：开关切换，正常/下架
- **是否置顶** (`isPinned`)：开关切换
- **是否推荐** (`isRecommended`)：开关切换

**底部操作按钮：**
- **保存草稿** (`POST /api/document/docVideo/draft`)：保存为草稿状态
- **确认发布** (`PUT /api/document/docVideo`)：更新视频信息并发布
- **取消：** 关闭弹窗，不保存修改

#### 3.1.3 表单校验规则

| 字段 | 校验规则 |
|------|----------|
| videoTitle | 必填，长度 1-100 字符 |
| broadCode | 必填 |
| fileId | 必填（必须有视频文件） |
| coverFileId | 必填（必须有封面图片） |
| tags | 选填，单个标签长度不超过 20 字符 |
| metaData | 选填，必须为合法 JSON 格式 |

#### 3.1.4 数据回显与初始化

- 编辑模式：根据视频 ID 调用 `GET /api/document/docVideo/{id}` 获取详情，回显所有字段
- 新增模式：所有字段为空，状态默认值为 `status=3`（草稿）、`auditStatus=0`（待审核）、`isPinned=0`、`isRecommended=0`
- 文件字段回显：根据 `fileId` 和 `coverFileId` 回显视频预览和封面预览

| 功能 | 后端API | 状态 | 说明 |
|------|---------|------|------|
| 查询视频列表 | GET /api/document/docVideo/list | 已支持 | 支持videoTitle、status、auditStatus、broadCode、narrowCode筛选 |

| 上传视频文件并保存 | POST /api/document/docVideo/upload | 已支持 | 支持multipart/form-data上传 |

| 删除视频 | DELETE /api/document/docVideo/{ids} | 已支持 | 支持批量删除 |

| 保存草稿 | POST /api/document/docVideo/draft | 已支持 | 保存视频草稿 |
| 发布草稿 | PUT /api/document/docVideo/draft/publish/{id} | 已支持 | 将草稿发布为正常状态 |
| 批量更新状态 | PUT /api/document/docVideo/status/batch | 已支持 | 支持批量上架/下架 |
| 切换置顶状态 | PUT /api/document/docVideo/{id}/pinned | 已支持 | 切换置顶 |
| 切换推荐状态 | PUT /api/document/docVideo/{id}/recommended | 已支持 | 切换推荐 |
