# [设计]菜单管理

**文件位置：** `src/pages/Admin/Personnel/Menu.tsx`  
**最后更新日期：** 2026-04-20

---

## 功能需求

### 0. 页面布局
- **布局方式：** 左右分栏布局
- **左侧区域：**
  - 菜单树展示，支持展开/折叠
  - 宽度固定，约占整体宽度的 35%
- **右侧区域：**
  - 二层结构布局
  - 上层：查询工具栏（搜索框、筛选条件）
  - 下层：菜单详情列表（表格形式展示）
  - 宽度自适应，约占整体宽度的 65%

### 1. 菜单树展示
- **功能描述：** 以树形结构展示系统菜单，支持展开/折叠
- **展示内容：**
  - 菜单层级关系（支持children嵌套）
  - 菜单名称
  - 菜单图标
  - 菜单类型（M菜单 C目录 F按钮）
  - 路由路径
  - 权限标识
  - 菜单排序
  - 菜单状态

### 2. 菜单列表查询
- **功能描述：** 查询菜单列表，支持筛选
- **查询条件：**
  - 菜单名称（模糊查询）
  - 菜单状态（0正常 1停用）
  - 菜单类型（M菜单 C目录 F按钮）

### 3. 用户权限菜单查询
- **功能描述：** 根据用户ID查询该用户有权限访问的菜单树列表
- **使用场景：** 侧边栏导航菜单渲染

### 4. 菜单管理
- **功能描述：** 管理系统菜单
- **功能包括：**
  - 新增一级菜单（parentId传0）
  - 新增子菜单（指定parentId）
  - 编辑菜单信息
  - 删除菜单（支持批量删除）
  - 拖拽排序菜单（调用批量更新接口）

### 5. 菜单编辑
- **功能描述：** 编辑菜单详细信息
- **编辑内容：**
  - 菜单名称（必填）
  - 菜单类型（M菜单 C目录 F按钮，必填）
  - 父菜单ID（parentId，顶级菜单传0，必填）
  - 菜单图标
  - 路由路径
  - 组件路径
  - 权限标识
  - 排序号
  - 菜单状态（0正常 1停用）
  - 备注

### 6. 图标选择
- **功能描述：** 提供图标选择功能
- **功能包括：**
  - 内置图标库
  - 图标搜索
  - 图标预览
