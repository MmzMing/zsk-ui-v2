# 后台-字典管理

**文件位置：** `src/pages/Admin/System/Dict.tsx`  
**最后更新日期：** 2026-02-16

---

## 0. 页面布局

### 0.1 整体布局
- **布局类型：** 工具栏 + 表格布局（字典类型列表）+ 子表格（字典数据列表）
- **顶部工具栏区域：**
  - 左侧：搜索框（支持字典名称、字典编码模糊搜索）
  - 中间：状态筛选下拉框
  - 右侧：操作按钮（新增字典）
- **中部表格区域：**
  - **上方**：字典类型列表表格
    - 表格列：字典名称（dictName）、字典编码（dictType）、字典状态（status）、备注（remark）、创建时间（createTime）、操作
    - 支持多选批量操作（批量删除）
  - **下方**：字典数据列表表格（选中字典类型后显示）
    - 表格列：字典标签（dictLabel）、字典键值（dictValue）、排序（dictSort）、样式属性、备注、操作
- **底部：** 分页控件（页码、每页大小、总条数）
- **弹窗组件：**
  - 新增字典弹窗
  - 编辑字典弹窗
  - 新增字典数据弹窗
  - 编辑字典数据弹窗
  - 批量删除确认弹窗

### 0.2 响应式适配
- 桌面端：完整展示所有表格列
- 移动端：隐藏部分非关键列，操作列自适应

## 1. 字典类型列表

- **功能描述：** 展示系统字典类型列表，支持分页、搜索、筛选
- **功能包括：**
  - 分页展示字典类型列表
  - 按字典名称、字典编码模糊搜索
  - 按状态筛选（正常/停用）
  - 列表字段：字典名称、字典编码、字典状态、备注、创建时间
  - 支持多选批量操作

---

## 2. 字典类型管理

- **功能描述：** 管理系统字典类型
- **功能包括：**
  - **新增字典**：填写字典名称、字典编码、状态、备注
  - **编辑字典**：修改字典名称、状态、备注
  - **删除字典**：支持单个删除和批量删除
  - **启用/禁用字典**：切换字典状态

---

## 3. 字典数据管理

- **功能描述：** 管理字典数据项
- **功能包括：**
  - **查看字典数据列表**：选中字典类型后显示其下的数据项
  - **新增字典数据**：填写字典标签、字典键值、排序、样式属性、备注
  - **编辑字典数据**：修改字典标签、字典键值、排序、样式属性、备注
  - **删除字典数据**：支持单个删除和批量删除
  - **字典数据排序**：调整字典数据显示顺序

---

## 4. 字典详情

- **功能描述：** 查看和编辑字典详细信息
- **编辑内容：**
  - 字典名称
  - 字典编码
  - 字典状态
  - 备注

---

## 5. 字典数据详情

- **功能描述：** 查看和编辑字典数据详细信息
- **编辑内容：**
  - 字典标签
  - 字典键值
  - 字典排序
  - 样式属性
  - 备注

---

## 6. 接口依赖分析

| 查询字典类型列表 | GET /api/system/dict/type/list | ✅ 已支持 | 支持dictName、dictType、status筛选 |

| 删除字典类型 | DELETE /api/system/dict/type/{ids} | ✅ 已支持 | 支持批量删除 |
| 查询字典数据列表 | GET /api/system/dict/data/list | ✅ 已支持 | 查询数据列表 |
| 根据类型查字典数据 | GET /api/system/dict/data/type/{dictType} | ✅ 已支持 | 按类型查询 |
| 获取字典数据详情 | GET /api/system/dict/data/{id} | ✅ 已支持 | 根据ID获取详情 |
| 新增字典数据 | POST /api/system/dict/data | ✅ 已支持 | 完整支持 |
| 修改字典数据 | PUT /api/system/dict/data | ✅ 已支持 | 完整支持 |
| 删除字典数据 | DELETE /api/system/dict/data/{ids} | ✅ 已支持 | 支持批量删除 |
| 切换字典状态 | PUT /api/system/dict/data/toggleStatus | ✅ 已支持 | 单个切换 |
| 批量切换字典状态 | PUT /api/system/dict/data/batchToggleStatus | ✅ 已支持 | 批量切换 |
