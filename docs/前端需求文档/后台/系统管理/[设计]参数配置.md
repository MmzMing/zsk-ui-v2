# [设计]参数配置

**最后更新日期：** 2026-02-16

## 0. 页面布局

### 0.1 整体布局
- **布局类型：** 工具栏 + 表格布局
- **顶部工具栏区域：**
  - 左侧：搜索框（支持参数名称、参数键名模糊搜索）
  - 中间：状态筛选下拉框
  - 右侧：操作按钮（新增参数、刷新缓存）
- **中部表格区域：** 参数列表表格
  - 表格列：参数名称（configName）、参数键名（configKey）、参数键值（configValue）、系统内置（内置标识）、备注（remark）、创建时间（createTime）、操作
  - 支持多选批量操作（批量删除）
  - 底部：分页控件（页码、每页大小、总条数）
- **弹窗组件：**
  - 新增参数弹窗
  - 编辑参数弹窗
  - 批量删除确认弹窗

### 0.2 响应式适配
- 桌面端：完整展示所有表格列
- 移动端：隐藏部分非关键列，操作列自适应

---

## 1. 参数列表

- **功能描述：** 展示系统参数列表，支持分页、搜索、筛选
- **功能包括：**
  - 分页展示参数列表
  - 按参数名称、参数键名模糊搜索
  - 按状态筛选（正常/停用）
  - 列表字段：参数名称、参数键名、参数键值、系统内置、备注、创建时间
  - 支持多选批量操作

---

## 2. 参数管理

- **功能描述：** 管理系统参数
- **功能包括：**
  - **新增参数**：填写参数名称、参数键名、参数键值、状态、备注
  - **编辑参数**：修改参数名称、参数键值、状态、备注
  - **删除参数**：支持单个删除和批量删除
  - **刷新参数缓存**：刷新系统参数缓存

---

## 3. 参数详情

- **功能描述：** 查看和编辑参数详细信息
- **展示内容：**
  - 参数基本信息（参数名称、参数键名、参数键值、状态、备注、创建时间、更新时间）

---

## 4. 接口依赖分析

| 功能 | 后端API | 状态 | 说明 |
|------|---------|------|------|
| 查询参数列表 | GET /api/system/config/list | 已支持 | 支持configName、configKey、status筛选 |
| 获取参数详情 | GET /api/system/config/{id} | 已支持 | 根据ID获取参数详情 |
| 新增参数 | POST /api/system/config | 已支持 | 完整支持 |
| 修改参数 | PUT /api/system/config | 已支持 | 完整支持 |
| 删除参数 | DELETE /api/system/config/{ids} | 已支持 | 支持批量删除 |
| **刷新参数缓存** | - | 缺失 | 需要后端提供刷新缓存API |
| **系统内置标识** | - | 缺失 | API返回中缺少系统内置字段 |
